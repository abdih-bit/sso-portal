## 🚀 Langsung Deploy (Untuk yang Impulsive)

Copy-paste command berikut ke VPS:

```bash
# SSH ke VPS
ssh root@YOUR_VPS_IP

# Go to folder
cd /www/wwwroot/portal.hqmedan.com

# Clone repository
git clone https://github.com/YOUR_USERNAME/sso-portal.git .

# Setup environment
cp .env.example .env
nano .env
# Edit dengan DATABASE_URL dan secrets

# Install & setup
npm install --omit=dev
npx prisma generate
npx prisma migrate deploy

# Verify
pm2 status
psql -U sso_user -d sso_portal -c "SELECT 1;"

# Test di browser
# https://portal.hqmedan.com
```

**Untuk detail tiap langkah:** Lihat AAPANEL_SETUP_GUIDE.md
