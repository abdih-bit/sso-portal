## 🚀 5-MINUTE QUICK START

```bash
# SSH ke VPS
ssh root@YOUR_VPS_IP

# Go to folder
cd /www/wwwroot/portal.hqmedan.com

# Clone & setup
git clone https://github.com/YOUR_USERNAME/sso-portal.git .
cp .env.example .env
nano .env  # Edit dengan database credentials

# Install
npm install --omit=dev
npx prisma generate
npx prisma migrate deploy

# Setup di aaPanel:
# 1. Supervisor → Add Process (npm start)
# 2. Websites → Reverse Proxy (127.0.0.1:3000)
# 3. Websites → SSL → Let's Encrypt

# Test
curl https://portal.hqmedan.com
```

**Untuk detail:** Baca AAPANEL_SETUP_GUIDE.md
